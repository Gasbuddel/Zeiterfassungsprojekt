Hilfe - Mitarbeiter

In diesem Reiter sehen sie alle Mitarbeiter in einer Liste. Sie können einen Mitarbeiter auswählen und sehen im unteren Bereich Detailinformationen. Sie können mit dem Button „Ändern“ diese Informationen ändern. Sie können mit dem Button „Neuer Mitarbeiter“ einen neuen Mitarbeiter hinzufügen. Sie können mit dem Button „Passwort zurücksetzen“ das Passwort auf den Standardwert #start12 setzen.

Sie können mit dem Button „Mitarbeiter entfernen“ einen Mitarbeiter löschen.
