Hilfe - Kunde

Hier sehen sie eine Liste aller Kunden. Sie können einen Kunden auswählen und sehen im unteren Bereich Detailinformationen.

Sie können mit dem Button „Bearbeiten“ diese Informationen ändern. Sie können mit dem Button „Kunde entfernen“ den Kunden löschen. Sie können mit dem Button „neuer Kunde“ einen neuen Kunden mit den entsprechenden Informationen eingeben.
