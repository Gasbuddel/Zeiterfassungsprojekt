Hilfe - Projekte

Im Dropdown-Menü „Projekt“ können sie das zu bearbeitende Projekt auswählen. Sie können mit dem Button „Projekt anlegen“ ein neues Projekt erstellen. Sie können mit dem Button „Bearbeiten“ die Projektbeschreibung ändern. Unter der Projektbeschreibung finden sie eine Liste der Projektmitarbeiter. Diese können sie über den Butto „Mitarbeiter entfernen“ im Projekt inaktiv schalten. Sie können mit dem Button „Mitarbeiter hinzufügen“ einen weiteren Mitarbeiter zum Projekt zuweisen.

Darunter finden sie eine Liste von Tätigkeitsvorschlägen. Sie können mit dem Button „Tätigkeit hinzufügen“ eine Tätigkeit auswählen die dem Projekt hinzugefügt wird.

Sie können mit dem Button „Neue Tätigkeit“eine neue Tätigkeit kreieren. Diese wird automatisch auch dem gewählten Projekt hinzugefügt Sie können mit dem Button „Tätigkeit entfernen“ eine Zuweisung entfernen.
